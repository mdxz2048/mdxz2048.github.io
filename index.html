关于疫情的一点思考

发表于 2020-02-08 更新于 2020-02-24 分类于 随笔
本文字数： 659

早在1月23日武汉封城之前，大约是1月10日左右，就看到一些关于武汉出现新型肺炎的消息，印象最深的消息就是这个病毒和SARS有关，当时以为是一些谣言没有在意，直到1月23日凌晨武汉对外宣布封城才意识到问题的严重性，紧接着就是其他城市一个接着一个的连锁反应，各省市都启动了紧急预案，尽可能对人流进行控制，到今天已经是在家里不出门的第15天了，在这里把最近的想法整理出来分享给大家：

作为一个普通公民，当国家遇到这种大的疫情时，关于疫情期间的一些处置方案，每个人都有服从和配合的义务，服从疫情期间的相关规定，具体做法通知了不让出门就不要出去瞎转悠，春节不允许走亲戚就待在家里不要出去，不能帮忙解决问题，也不要主动给政府添乱。另外，作为疫情的重灾区的湖北人民，他们也同样是这次病毒的受害者，我们要对他们有保持充分的理解和关心，但同时也要做好科学的防护措施。

作为一个互联网的使用者，大家可能会在网上看到很多来路不明的消息(如聊天截图、语音、某专家语录)，我们普通用户经常无法判断消息的真实性，但是我们可以做到不随便相信和传播不实言论，此时，官媒和公众媒体才是最可靠的消息来源。这次疫情也是公众对官媒信任程度的一次考验，也真的想看到我们官媒能更快速、真实的向公众传播疫情真相。

作为一名普通上班族，在这种特殊时期，有远程办公条件尽可能选择远程工作；必须返回公司上班的岗位，在保证个人安全的前提下，尽可能回到工作岗位保障工作，这也是对作为普通上班族个人职业素养的一种考验。

疫情已经发生，无论是天灾还是人祸，都需要我们所有人的齐心协力。

我们终究要尊重点什么吧

上边这张图片来自于11月1日日的一则社会新闻— 浙江省金华市孝顺镇中心小学在课堂给小学生带头环监测学生上课专注度。

每当看到这种新闻的时候，我是相当恐惧的，真不知道我们这个社会到底要去往哪里…，我们这么小的受教育群体，竟然要通过这种几乎是管监狱犯人的方式来管理，不客气的说这已经是在侵犯人的隐私，或者说是在践踏生命的尊严，再说难听点，这和我们给宠物狗带上铃铛有多少区别？我真的不能接受这是一个学校能做出来的事情，无论是出于什么目的。

我们需要进步，需要更先进的教学方式，可是我相信更好的教育的前提一定是尊重思想、尊重生命吧。

之前和朋友谈过关于生活中监控摄像头、网络监控的话题，其实和这个头环的性质差不多，只不过那个监控给人直观的冲击力没有这么强烈，但其实是范围更大，让人无处躲藏。打个比方，就像人穿衣服一样，作为一个人有穿衣服和不穿衣服的权利，但是如果剥夺了身而为人的选择权利，那我们离机器又近了一步…

不知道有多少人了解“雪梨”事件，就因为一个女生直播中没有穿内衣就受到了网友的网络暴力，最终二十几岁的女生无法承受选择了终结自己的生命。这件事之后我这几天一直在想，我总觉得会有一个突破口能避免这一类的事情再发生，或者促进我们远离这些，我想过“尊重女性”，“尊重生命”，“尊重思想”，”人性、自由“，我不知道最终的答案是什么，但是这一类事件之后，我们总得有所思考吧。

我知道自己的声音太小，这并不重要，重要的是抓住了表达态度的机会，留下点文字给自己，等某一天回头看的时候，不至于因为当时自己的无动于衷而懊恼，毕竟我们总要走向更高级的地方。

一个理工科本科毕业生的就业经历分享

发表于 2019-09-15 更新于 2020-02-24 分类于 生活黑客
本文字数： 2.2k

如果你有兴趣打开了这篇文档，我大概可以相信你现在也是处于大四刚开始找工作的阶段，写这篇文档是因为最近身边有一些学弟学妹来问我关于找工作的事情，就正好利用周末一点时间结合自己参加校招和社招的经历，分享一点点经验给大家。有一点要说明的是，作为我个人的一些经验分享，并不一定适用于当下找工作的形势与政策，也一定不是适用于每个人，但是对于正在或者即将面临毕业的你，我相信或多或少的会有一些有参考价值。

我是自动化专业19届本科生，现在在深圳从事嵌入式计算机行业，毕业前掌握一些单片机和嵌入式linux方面的知识，有一定的C语言基础，另外还有一些实际的学生项目经验，所以我以下的分享可能更适用于有一点技术但是没有选择考研的同学作为参考。

以下我按照我找工作的安排顺序，分为校招、和社招两部分来给大家分享：

校招：

首先是关于招聘会的选择，一开始我就比较重视也算积极的参加了校招，我是在秋季学期的9月和10中旬一共参加了两次。因为考虑毕业后打算去北京，所以当时我选择了在西安一些高校参加校招(之所以没有选择北京完全是因为我对西安更加熟悉一些)

宣讲会信息的整理，参加校招需要提前了解各个学校的宣讲会信息，提前安排好行程，每个学校的校招信息在各个学校的就业信息服务网站中的宣讲会模块都会有，为了保证信息的准确性和实时性建议大家去各高校的官网去看，我会将网址附在文末。为了自己少跑一些路，我将每天的招聘信息按照各个校区(一定注意是校区，有一些学校两个校区之间可能隔两三小时车程)进行分类整理，然后再将同一时间段内(一般就是一个上午或者下午)自己想要参加的招聘信息放在一起，当时是做了表格方便自己随时查看。

其次是校招过程自己总结出的一些经验：

  1. 因为很多一场招聘会的时间都在1-2个小时，所以基本上同一间教室一个时间段内一般只有2-3个宣讲会，所以可能你在这所学校感兴趣的两个宣讲会会同时在同一个时间进行宣讲，所以为了应对这种问题建议不是在本校找工作的同学尽量结伴。

  2. 关于简历的制作技巧就不在这里说了，在职位这一块我建议写成就业方向，然后选择一两个从事的方向去写，因为每个企业对岗位的叫法不一样，到时候可以在简历明显的位置用笔写上岗位就可以，这里也避免了让人事小姐姐懵O。

  3. 校招的流程一般不确定，笔试、面试一般都或前或后的都有可能(如果你有机会参加的话)，这个时间也很不确定，有的是当场进行，有的是电话通知，一般这个就听安排然后按照自己的选择进行安排就行。面试也主要是问一些简历和专业方面的问题，大部分都比较基础。

社招：

对于社招，我首先想抛出我的一个观点：对于第一份工作，企业给你的成长空间比待遇会更重要一些，应当在想要从事的行业去尽可能找一些更大的平台去提升自己。(当然这个目标是在待遇能满足你正常生活的前提下，毕竟这是一件长期的事情，总得每天开开心心的去工作)
社招的准备：

  1. 简历一般建议多打印几份，普通黑白打印即可，基本上每家去之前都要求带简历。
  2. 行程安排我这次主要使用手机的日历功能，手机会会提前提醒，也是方便安排行程。
  3. 基本上大的企业面试都是一些基础的问题，我自己是做软件，做的面试题也基本是一些C语言(主要是指针、内存、常用接口、链表…)相关和三电起的基础；问题也主要是针对简历上所写的来问，基本上只要是自己做的东西一般也不会被问住。

公司的选择：

我有在北京一家公司待过一段时间，公司主要研发港口自动化方面的系统，规模在一百人左右，其实规模已经不是特别小了，虽然进去也是研发岗位，但是由于一些管理制度上的不完善，导致工作体验比较差，可见范围内也不能很好的提升自己，因此我选择了在答辩完后去深圳找工作，大概是在校运会的那一周我在深圳参加面试，相对来说深圳研发环境比北京好一些，所以我也是很快在深圳找到了工作，但是北方的同学来深圳一定得考虑好南北气候的差异是不是能接受。

另外，如果是做技术方面的同学，尽可能去争取研发岗位，哪怕刚进去是在研发岗干的是助理的活。岗位不一样想转岗难度一般会比较大，除非你是特别优秀或者选择跳槽。

平台的选择：

我一共用了三家招聘平台的APP：BOOS直聘、前程无忧(51Job)、智联直聘，现在看来后两家企业的回复会多一些，boos直聘更多的是企业主动联系你让你发简历。

写在最后：

现在正处于九月中旬，也就是秋招刚刚开始的阶段，大家完全有时间去多参加一些面试，无论是否成功给自己积累一下经验总不是件坏事。作为本科毕业生想要从事技术领域相关工作，更多的还是要真正学到一些知识、技能，这也是你找工作的资本。如果你到现在还是一张白纸什么也不会，那么以上的内容可能并不适用。

最后，真诚的希望各位能够珍惜大学最后的这段时光，尽可能到处走走，多和朋友、老师在一起聚一聚。

个人水平有限，小小的分享只希望能够帮到一二就相当满足了，仅作为参考，不建议生搬硬套。

又一个周末就这样过去了，从学校出来已经过了整整两个月了，从刚进公司的那种小心翼翼，到现在每天基本上很积极的数日子盼周末，到周末又是浑浑噩噩，基本上周内对周末的幻想都无疑会被所有的懒散打败，就像所有在学校对工作的憧憬也迅速被每天平平淡淡的上下班所击破一样。

最近情绪波动其实特别大，一个是因为听了一些比较感动的歌曲和故事，在还有就是看到了自己和身边很多刚毕业的同学们的一些近况，实在不能让人心情舒畅。但因为实在太懒，就没有很及时的记录下来。

可能是由于刚换了一种生活方式吧，所有人都或多或少的需要一段的时间来整理一下自己的心情，用一个正确或者说是成熟一点的心态去面对自己新的这种生活，多年来的校园生活这下真的真的就说了再见， 而且又这么突兀、生硬的直接进入了工作岗位，这其中的一些慌张、恍惚、迷茫、胆怯、挫败可能只有刚毕业的我们这些小朋友们才能会有深切体会吧，最近听到了大概五六七八个同学都在工作方面遇到了比较不好的状况，无关乎能力，其实就是刚毕业的仓促感让人没有空隙去在很多的压力中去学会喘息，我们就这样匆匆忙忙的告别，然后很仓促的做了人生这个可能会比较重要的一个选择，距离答辩完已经过去了91天了，很久没有这种在短时间内涌出如此复杂的情绪，甚至已经超过了我可调节的范围。

现在最希望的就是我那些可爱的朋友们，能够找到一个好的环境开始接下来的生活，并且多年后相见的时候，大家能够依然可以可爱。

还有一段也想记录下来，是上周末我从深圳坐车去了东莞，因为在那里有我好两个个比较好的朋友，其中就有我大学室友(还没有这么给我麻哥想过一个称号，反正聊得特别好的一个朋友了)。他比我晚来几天这边，我是周六去的东莞，去的当天那边上午刚下过暴雨，我们在一段在坑坑洼洼都是水潭的路上碰面，破旧的东莞马路！（原谅我用了破旧，可是我去的那个地方给我的感觉就是这样。）让我美好的幻想一下破灭，不过这似乎也比较符合我们当下的心情。毕竟去之前我是想着看电影，吃烧烤，吃龙虾，在干点其他反正能让人放松的事情。当我们俩坐在桌子对面吃饭的时候，真的有一种感觉就是互相都不舍得说太多丧的话担心影响到对方的情绪，因为大家确实都感受到了这种未曾有过的惶恐，他经常跑去公司加班学东西，可能也是因为想多学一点、也可能是出租屋里太热。能做的只能是互相鼓励，或者是开玩笑，或者是其他....

刚毕业的大家就是这样，我们都不知道自己适合去往哪里，一个个青涩的年轻人从大学的乐园里走出来，褪下了之前学校里各种颜色的保护壳，挤进了这个看起来混乱不堪的社会，在这个巨大的齿轮上各自找寻着自己感觉合适的位置，每个人心里都有着各自的酸楚，就这样吧，希望大家一切安好。

突然觉得那句祝“工作顺利”是一句多么有分量的话！​

12306候补制度

发表于 2019-07-25 更新于 2020-02-24 分类于 生活黑客
本文字数： 640

这两天需要买一张火车票，但是打开12306APP时候发现已经卖完了，就想着是不是要抢票软件去买加速包了，但是看到12306提示的是 候补 而不是无票，就去了解了一下这个候补的规则，真香！

首先大概说一下没有候补之前售票流程:所有官方放出来的票会直接进入一个 票池 ，12306官网、客户端以及所有抢票软件都是在这个票池中买票，当没有票的时候其实大家都买不到票，但是一发现有人退票，因为软件可以一直监测票池，所以就比人有优势抢到票，当然这是 以前 。

再来看看现在有了候补制度的售票流程:因为存在候补，在放出的票 进入票池之前 会先查看是否有候补订单，如果有直接根据候补订单的下单时间顺序（注意这里是直接根据订单的提交时间顺序！）将票分配给对应的订单，这样这张票就已经售出不会进入票池，当没有候补订单时才会进入票池，接下来才是各类APP和后面的乘客继续购买。

当然候补终止时间为“不晚于开车前一天的19:00”，所以在这个时间节点之后候补通道关闭，也就是开车前一天的七点之后不可以进行候补，如果这时候还没有买到票就需要去票池等别人退票了，这个时间到开车前停止售票这个时间段才是12306给抢票软件留下的生存空间。

现在我们可以分析一下，候补通道有效时，抢票软件其实是在这个候补订单之后买票，相对于这个候补机制完全没有了优势。那么我们买票完全可以直接在12306官网进行购买或候补，成功率明显高于抢票软件，而且不用去到处给别人分享加速包。而在候补通道关闭后剩下那一段时间内，你可以尽情去用你喜爱的抢票软件了。

应届生第一份工作做的怎么样了？

发表于 2019-04-22 更新于 2020-02-24 分类于 随笔
本文字数： 635

4月7号那天，我推着行李从学校坐火车来到北京，开始了毕业前为期一个月的实习生活。到今天仅仅过去了半个月，但是这两天一直有一种强烈的感觉想写一些东西，正赶上今天中午休息时间，找了个地方打开电脑就随便写一下了。

实习的公司是在北京一家做港口仪器(担心被看到，就不具体写干什么的了)，从8号上班开始，在北京待了三天，也没有交代具体的事情，就是开了两天会让我熟悉一下公司业务。随后开始出差，估计回学校之前都会在一直出差吧，出差这段时间确实是接触到了公司的一些产品，也帮着给调了一些代码，带我的工程师也很信任的把手里在调试的程序给了我，看了一下，也是不少人在维护的其中不缺乏各种版本的祖传代码。看了师傅给我准备的培养计划，说实话不是很感兴趣，可能是之前面试的时候了解不够吧，感觉公司后期的需求主要还是维护，而且维护中涉及到的一些技术也是之前我在学校已经早早接触过的，和我以后想要从事的行业和领域都存在较大的差距。

今天距离实习结束还有将近一周的时间，上班这段时间，公众号也没有时间更新了，因为每天要看代码、测试程序、回邮件、写论文，还得在现场各种跑，幸好抽时间将我的博客平台搭建起来了，这是这段时间我最开心的事情了。
第一次出来工作，感觉其实比较纠结，至于毕业后是回到这家公司还是另找出路，谁知道呢！
一上午都在干活，也不知道写了点什么就这样发了吧，最后打个广告。
